
## Iniciando a API

Abra o terminal na pasta backend e rode os seguintes comandos.

Caso utilize o NODE:
```
npm install
npm start
```
Caso utilize o YARN:
```
yarn
yarn start
```


## Iniciando a Aplicação

Abra o terminal na pasta application e após instalar o python rode o seguinte comando.

```
python client.py
```
